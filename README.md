# Smart Fan

一个人体跟随风扇项目，包含 STM32 固件、微信小程序控制端，以及部分风扇结构建模文件。

## 项目结构

- `stm32风扇代码`
  STM32 端工程，负责风扇控制、传感器采样、串口/BLE 数据发送等。

- `微信小程序开发/BLE调试助手`
  微信小程序控制端，支持 BLE 扫描连接、风扇控制、场景模式、定时关闭、实时状态同步。

- `风扇建模`
  风扇外壳和结构建模相关脚本与三维文件。

## 当前功能

### 微信小程序

- BLE 设备扫描与连接
- 风扇开关控制
- 普通模式、摇头模式、人体跟随模式
- 1 到 5 档风速控制
- 场景模式
- 定时关闭
- 智能模式
- 首页风扇速度显示
- STM32 回传数据实时同步
- 实体按键变更后的功能同步

### STM32

- AS5600 转速读取
- OLED 转速显示
- 串口/蓝牙数据包发送

## 通讯协议

小程序当前兼容两种数据包格式：

```text
FF 温度 湿度 风速 FE
FF 温度 湿度 风速 同步码 FE
```

### 同步码说明

- `00`：开机
- `01`：关机
- `02`：普通模式
- `03`：摇头模式
- `04`：人体跟随模式
- `05`：1 档
- `06`：2 档
- `07`：3 档
- `08`：4 档
- `09`：5 档

当风扇实体端通过按键修改开关、模式或档位时，只要 STM32 按上面的同步码回传，小程序页面就会自动同步状态。
